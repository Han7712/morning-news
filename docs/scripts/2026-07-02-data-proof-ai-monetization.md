# Morning News 2026-07-02 Script

## 播客正文

大家好。现在是香港时间7月2日，周四中午。今天的主线先放在这里：二季度那波risk-on没有被一夜推翻，但市场已经从追涨切到要证据。美股新季度第一天高开低走，Dow只跌0.03%，标普500指数跌0.22%，Nasdaq跌0.66%；ADP显示6月私人就业只增加9.8万人，ISM制造业还在扩张，只是速度慢一点，今晚的非农就成了利率和美元的下一道坎。Warsh在Sintra继续讲2%通胀目标，也不给市场提前答案。公司端，Meta靠一条出售多余AI compute的报道大涨8.8%，CoreWeave和Nebius反而跌得很重。亚洲端，日元卡在162附近，40年低位压力还在；中国PMI扩张，但出口订单和国内需求温度不同。今天要看的，是这轮风险偏好能不能被数据、现金流和融资簿记接住。

先说美国盘。表面看，跌幅不大。Dow离纪录高位很近，标普500指数和Nasdaq只是从二季度很强的收官里回头喘口气。可盘中结构比指数更有用：早盘还有轮动，金融、小盘和部分周期股接住了一点资金，下午技术股和半导体拖着盘面往下。TheStreet给出的收盘数比较清楚，Dow收在52305点附近，标普和Nasdaq都翻绿，Russell 2000盘中创新高后也收跌。换句话说，资金没有全面撤退，但不愿意继续无条件买高估值。

这个小回撤值得讲，因为它正好卡在两个节点上。二季度刚结束，标普和Nasdaq前一晚交出多年少见的强季度表现，很多仓位已经有利润；周五美国因为独立日补假休市，债市周四还会提前收盘，流动性会变薄。这样的日子，短线仓位管理很容易被误读成大方向变化。我更愿意把昨晚当成一次质量检查：谁能在获利盘出来时稳住，谁的上涨还停留在叙事上。

利率这边，ADP和ISM放在一起看，信号有点拧。ADP说6月私人部门新增就业9.8万人，低于5月的12.2万人；工资同比涨幅对留任员工是4.4%，跳槽员工是6.6%。这不是崩盘式就业，但招聘速度确实慢下来，休闲酒店业连续第六个月几乎没什么增量。ISM制造业PMI是53.3，连续第六个月扩张，新订单还在56；问题是价格指数仍有73，虽然比5月的82.1降了一截，企业评论里仍然很多人提到伊朗战争、关税和价格波动。对Fed来说，这不是可以闭眼宽松的数据。

Warsh在ECB论坛上的信息也很直接。他承认通胀风险比上次会议有所下降，但继续守住2%目标，也不愿意给7月利率决定写小抄。债券市场的反应是先松一下，又很快回到数据前的紧张感。MarketWatch写到10年期美债收益率升到4.46%附近，离伊朗战争前的一年低点已经高了接近50个基点。这里的交易含义很实际：今晚非农如果只是温和放缓，市场未必能重新定价降息；如果就业明显断档，美元和长端收益率才会一起松。

还有一个小细节别忽略。周四美国债市提前收盘，周五股债都休市，很多宏观基金会在非农前先把杠杆降一点。数据公布后的第一反应，可能比平时更容易被流动性放大。做盘时要把这个因素放进止损和成交量判断里，不要只看一个点位跳动就改中期观点。

所以今天的美股主线别简单贴上risk-off标签。买方更像是在问：二季度的估值扩张，有没有足够的利润、就业和通胀证据继续支持？对hedge fund，指数仓位可以轻一点，但更重要的是做分化。金融、工业和小盘能不能接棒，取决于收益率别再冲太快；高duration科技股要继续赢，得拿出现金流和订单。

这就接到AI。Meta这条新闻很有代表性。Meta本来是Facebook、Instagram和WhatsApp背后的平台公司，过去两年为了AI买芯片、建数据中心，capex压力一直是市场心病。Reuters援引Bloomberg报道说，Meta正在研究把多余的AI算力做成云服务卖给外部开发者，可能让别人付费调用它的模型和基础设施。Meta当天收涨8.8%，盘中一度更高；CoreWeave和Nebius这些neocloud公司跌幅很大，因为投资者担心大客户Meta自己下场卖算力，同时减少向外部供应商采购。

这条线的重点不在于Meta明天就能挑战AWS、Azure和Google Cloud。AI capex的叙事正在被重新定价。过去市场愿意先给硬件链条估值：GPU、内存、设备、数据中心，只要和AI相关就有资金追。现在问题更直接，花出去的钱怎样回到收入表。Meta如果真能把闲置算力出租，市场会把它看成capex回收路径；如果做不起来，投资人还是会回到老问题：这么大的AI投入，到底什么时候转成利润。

半导体也有同样的压力。Axios统计说，费城半导体指数二季度涨了接近88%，速度很夸张。这样的涨幅后，任何一点订单、毛利率或客户支出节奏的变化，都会被放大。昨晚Nasdaq的弱势和芯片股波动，说明这笔AI交易开始进入审计阶段。做long/short的时候，我会少看一句“AI需求很强”，多看三件事：谁真的拿到长期订单，谁的客户愿意预付或签take-or-pay，谁的毛利率能扛住新产能上来后的价格压力。

再看亚洲。日元是今天亚洲资产里最扎眼的价格。WSJ报道美元兑日元触及162.8附近，是40年低位区间，日本财务大臣继续释放干预信号。对全球市场，日元弱不只是日本自己的问题。它会影响日股外资回报、亚洲出口商定价、套息交易，以及日本如果真干预时对美债储备的处理。现在日股本身还在高位，AI和半导体链条给了日本权益很强支撑，但汇率越往下走，政策风险越大。亚洲早盘如果看到日元突然拉升，很多cross-asset仓位会先降杠杆。

中国这边，PMI给的不是单一答案。官方制造业PMI从5月的50.0升到50.3，AP把它和AI相关出口联系在一起；RatingDog和S&P Global的私营制造业PMI是51.7，比5月51.8略低，但仍在扩张。细项更有用：国内新订单还不错，出口订单连续第二个月下滑，投入成本压力缓和，就业也有改善。市场可以把它读成制造业仍有韧性，但这份韧性更多来自外需和高端制造，内需和地产链条还没给出很强确认。

这对港股和A股的含义很直接。7月1日香港休市，今天重新开盘，市场要一次性消化美股回落、日元低位、美元利差和中国PMI。若资金只买AI出口链，那行情会很集中；若PMI里的就业改善和订单扩散能被后续数据确认，周期、机械、运输和部分金融才有机会跟上。IB角度也一样，给中国制造企业写材料时，不能只说“PMI扩张”，要说明客户来自哪里、出口订单有没有压力、毛利率是不是受益于成本下降。

最后讲香港IPO窗口。SCMP这周提到，香港今年的IPO热度被中国科技和A-to-H上市带起来，但Nasdaq仍然因为SpaceX这类超大交易占了全球注意力。HKEX自己的Q1回顾也显示，香港一季度有40家新上市，筹资超过1100亿港元，TMT和AI价值链是主力。这个背景放到今天看，很重要。因为二级市场如果开始要求AI现金流，一级市场也会同步提高门槛。发行人可以讲技术和成长，但簿记时投资者会问得更具体：客户是谁，收入能不能重复，上市后流动性够不够，估值折价给得是否诚实。

今天我的交易框架就这么收。第一，美股还没大幅转空，但二季度追涨结束后，市场开始要数据和利润证据。第二，今晚非农比昨晚ADP更关键，温和放缓不一定够鸽，明显走弱才会真正压美元和长端收益率。第三，AI还在主线里，可主线已经从买硬件动量，转向谁能把AI capex变成收入。第四，亚洲别只看日股强不强，日元、China PMI和香港IPO簿记质量，才是今天判断risk-on能否扩散的三条线。

如果要落到一个最实际的早盘动作，我会先看美元兑日元有没有触发干预式波动，再看美债10年能不能守住4.5%下方，最后看港股科技和券商是否同涨。如果只有少数AI硬件股涨，那还是拥挤交易；如果金融、交易所、工业和优质发行人一起被买，市场才是在告诉你，风险偏好真的从二季度动量，走向了三季度的基本面验证。
