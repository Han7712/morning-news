# Morning News 2026-06-11

## Spoken Script

今天的主线是，CPI和油价把市场从AI涨幅争论拉回通胀和资金成本：美股现金盘进入risk-off，Oracle盘后证明AI infrastructure订单仍强但更耗现金，中国PPI、日欧央行压力和亚洲股市回撤一起提醒我们，全球资产正在重算duration、能源传导和外部需求质量。

先说美国。昨晚最关键的数据是CPI。BLS公布，5月headline CPI环比涨0.5%，同比涨4.2%，比4月的3.8%又高一截；core CPI环比只涨0.2%，同比2.9%。表面看，core还算克制，市场一开始也能找到一点安慰。可真正麻烦在能源项。能源价格5月环比涨3.9%，BLS说它贡献了月度headline涨幅的六成以上，汽油单月涨7.0%，过去12个月涨40.5%。

这组数字给市场的信号很直接：Fed可以说核心通胀没有失控，但它很难在能源冲击还在传导时急着放松。更何况前几天强非农已经把higher for longer重新放回桌面。现在的交易问题从今年还会不会降息，转到如果油价和headline通胀把消费者预期拉上去，市场需要给多长的duration、多少的equity multiple打折。

美股现金盘给了答案。AP的收盘数据显示，标普500指数跌1.6%到7266.99，道指跌953点、跌幅1.9%，Nasdaq跌2.0%，Russell 2000也跌1.1%。这和前一天那种指数弱、内部还能轮动的盘面已经不同。昨天少数AI股继续被获利了结，工业、材料、科技和小盘也在承压，油价因为美国和伊朗紧张升级走高，市场先把风险预算缩回来。

对hedge fund来说，今天不能只问要不要抄底科技股。更好的问题是，组合里哪些头寸同时暴露在三件事上：高估值、长duration、能源或融资成本上行。如果三项都占，CPI偏热时会被卖得更快。反过来，现金流近、定价权更清楚、或者能从能源价格中受益的资产，才有机会在风险下降温后被重新挑出来。

债券市场其实没有崩。10年期美债招标的结果还算稳，4.538%的stop-out yield基本贴近当时二级市场，bid-to-cover在2.57，高于近六个月均值。这个细节很重要：买方没有拒绝美国duration，但它要求更清楚的通胀补偿。美债给出的含义更像是，资金还在，便宜的钱少了。IB角度也一样，融资窗口还在，可项目估值、债务成本和投资人筛选会更严格。

第二条线是盘后Oracle。Oracle是企业软件和云基础设施公司，过去一年市场越来越把它看成AI infrastructure平台，因为大客户会把模型训练、数据库和云算力需求放到Oracle Cloud Infrastructure上。昨晚它交出的收入和EPS都比预期好：季度收入大约191.8亿美元，调整后EPS是2.11美元，cloud infrastructure收入大约58亿美元，增速接近93%，RPO升到6380亿美元。

如果只看订单，这是很强的AI需求证明。但股票盘后下跌5%到6%，原因也很清楚：AI infrastructure越来越像重资产生意。MarketWatch和IBD都提到，市场担心Oracle的capex、负free cash flow以及未来还要通过债务和股权融资来扩数据中心容量。也就是说，AI订单可以很大，投资人现在会追问每一美元订单背后需要多少GPU、多少电力、多少折旧、多少融资。

这就是今天AI交易的材料增量。过去几天市场一直在卖拥挤的AI和半导体，Oracle给了一个更具体的框架：需求还在，问题落到ROIC和资产负债表。对long-only来说，AI链条里轻资产、现金流强的环节会更吃香；对credit和equity capital markets来说，数据中心扩张会继续制造融资需求，但融资会带上更高的审查。Oracle如果要把RPO兑现成收入，资本市场要先愿意给它足够长的时间。

对软件股也要小心。过去软件公司的估值享受的是高毛利、高可见度和低资本强度。AI cloud把可见度拉高，却把资本强度也拉高。投资人会开始区分两类公司：一类是用AI提高软件ARPU和留存，一类是为了AI收入被迫重资产化。估值方法会不一样，前者看margin和net retention，后者要看capex、折旧、融资成本和客户预付款。

第三条线看中国。5月中国PPI同比涨3.9%，是接近四年来最快的工厂端通胀，4月还只有2.8%；CPI同比1.2%，月度还小幅回落。这个组合听起来有点矛盾：上游价格热，下游消费还是温。它说明外部能源和原材料价格正在推高生产成本，但国内需求没有强到能顺畅传给消费者。

这对中国资产是双刃剑。好的一面是，通缩压力缓解，工业企业收入名义增速可能改善，尤其是能源、原材料、设备和出口链。坏的一面是，如果PPI上来主要靠油气和进口成本，margin反而可能被挤压。中国企业能否涨价，取决于终端需求和行业供给纪律，而不只是PPI数字本身。

再叠加贸易数据，画面更完整。AP和其他报道引用海关数据说，中国5月出口同比增19.4%，进口增27.4%，高于4月，科技、汽车、半导体和AI相关产品都强。这个数据让亚洲供应链还能撑住市场叙事：全球AI buildout没有停，订单还在跨境流动。但它也把问题推到更尖的位置。出口强可以稳增长，进口价格和PPI上行会压利润，贸易顺差继续扩大也容易引来欧美政策压力。

所以今天看A股和港股，我不会只用一句外需好来判断。更实际的是分三层：第一，真正有订单和议价能力的硬件、设备、工业品；第二，受进口能源和原材料挤压的中游制造；第三，消费和地产相关，它们还没有从CPI里看到强复苏。China trade能给市场底气，但China consumption还没有给同样强的确认。

第四条线是日欧央行。欧洲央行今天要开会，市场预期存款利率可能从2.0%升到2.25%。日本这边，Bank of Japan Governor Ueda住院，将缺席6月15到16日的会议，不过市场仍然广泛预期BOJ会加息到1.0%。这个组合说明，能源冲击正在逼几个主要央行从观望转向防守。

这里的市场含义不小。过去一年很多风险资产靠的是一个共同假设：美国可以soft landing，欧洲和日本也能慢慢正常化，全球利率压力逐步往下走。现在油价和headline inflation把这个假设弄得更脆。ECB如果加息，欧洲duration会重新定价；BOJ如果加息，日元和日债会影响全球carry；Fed虽然还没有开会，但美国CPI已经让降息叙事更难讲。

亚洲盘尤其要看日本和韩国。日本股市前期涨得多，日元又在160附近这种敏感位置，BOJ沟通稍微不顺，股债汇会一起波动。韩国的KOSPI最近因为AI和半导体仓位太满，波动也明显加大。这里更像同一条全球交易链：AI订单还强，油价和通胀也强，央行没法很快救估值。

把今天几条线合起来，我会这样处理：第一，CPI之后，美股的反弹需要先过油价和10年美债这一关；第二，Oracle说明AI仍然有需求，但AI infrastructure会被当成capex和financing trade来审；第三，中国数据支持外需和硬件链，暂时还不支持全面消费复苏；第四，ECB和BOJ提醒我们，全球利率环境可能比上周更硬。

今天早上的交易不适合太快恢复满仓risk-on。可以找结构性机会，比如能源链、现金流更近的工业和医疗、能证明订单质量的亚洲硬件；但对长duration growth和靠故事融资的项目，要把折现率调高一点。市场昨天真正说的是，AI没有结束，可便宜资金和宽松预期没法同时托住所有故事。
