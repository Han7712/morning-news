# Morning News 2026-06-12 Script

## 播客正文

大家早上好。今天先抓主线：伊朗风险降温让美股从通胀冲击里反弹，油价回落给 risk-on 一口气；但BLS的PPI和ECB加息提醒市场，能源冲击已经穿过企业成本，rate relief还没有真正回来。资金端同时被SpaceX的750亿美元IPO和Adobe盘后反应检验：稀缺叙事能拿到钱，现金流资产还得证明增长、治理和资本纪律。

先说隔夜最重要的价格。美国三大指数大涨，标普500指数涨1.8%，道指涨929点，Nasdaq涨2.5%，小盘Russell 2000涨3%。这次反弹不能理解成投资者忽然忘了通胀。更直接的触发点是特朗普取消对伊朗的新一轮打击，说谈判推进到了伊朗最高层。油价随即回落，Brent一度跌到每桶90美元附近，WTI收在87美元多。AP的收盘报道也提到，美债收益率明显下行，10年期从前一天的4.55%降到4.45%。

这对今天早上的盘面很重要。过去几天，市场担心的是同一条链：霍尔木兹压力推高油价，油价推高CPI和PPI，通胀再把Fed逼向higher for longer，最后压住长久期股票和融资窗口。昨晚的交易把这条链先松了一扣。油跌，利率跌，小盘和半导体反弹，风险预算开始补回。但我会把它看成地缘风险溢价的回吐，还不是通胀问题已经结束。伊朗方面还没有把美国的说法完全坐实，航运和能源流量也还需要继续观察。

第二层是数据和央行。BLS公布的5月PPI很硬，最终需求价格环比涨1.1%，同比涨6.5%，是2022年11月以来最大的12个月涨幅。更麻烦的是结构。最终需求商品价格涨2.8%，能源价格涨10.7%，汽油价格涨23.4%。剔除食品、能源和贸易服务后的指标环比涨0.8%，同比涨5.1%。这说明企业成本不只是油价标题那么简单，运输、化工、金融服务、库存和中间品都会进到利润率里。

劳动力数据没有给市场太多衰退安慰。美国初请失业金到22.9万人，比前一周多4000人，继续申请人数接近180万人。这个水平算不上过热，也算不上明显转弱。对Fed来说，这意味着它可以等，但很难因为一晚油价回落就转向宽松。欧洲那边更直接，ECB把存款便利利率提高25个基点到2.25%，主要再融资利率到2.40%，边际贷款利率到2.65%，并且说不会预先承诺利率路径。换句话说，欧美央行今天面对的共同问题是，增长开始受伤，通胀却还在企业成本端咬人。

这里的交易含义很实际。昨晚股市涨，是因为最坏的能源尾部风险被削掉一点；但融资成本还没有回到舒服区间。债券投资者今天要分清楚，收益率下行来自油价回落，还是来自真实增长恶化。股票投资者也要分清楚，小盘和半导体反弹是short-covering，还是有持续基本面买盘。IB角度更直接：高收益债、LBO模型、ECM窗口，都还要用更高的成本假设，不要因为一晚risk-on就把折现率调得太快。

第三件事是SpaceX。这个公司做火箭发射、Starlink卫星互联网，也把xAI和空间AI基础设施故事放进了上市叙事。市场关心它，马斯克个人标签是一层，更大的吸引力来自它把卫星通信、国防合同、AI算力和资本市场的想象力绑在一起。Reuters说，SpaceX以每股135美元定价，卖出5.5556亿股，募集750亿美元，估值约1.77万亿美元，成为史上最大IPO之一，周五将在Nasdaq交易。AP补充的公司逻辑是，SpaceX需要资本去部署下一代Starlink卫星和空间AI data centers，同时也会接受更强的公开市场披露和监管。

这是一条很强的ECM信号。过去几天市场一直在问，AI capex和高估值还能不能继续吸钱。SpaceX给出的答案是，稀缺资产和极端叙事仍然可以打开大票簿。SEC文件和AP解释稿也提醒我们，公开上市以后，它要接受季度披露、公众股东和治理结构的审视。这对IB来说意味着两个点。第一，2026年的IPO pipeline可能被重新打开，OpenAI、Anthropic这类AI相关公司会拿SpaceX作为定价锚。第二，治理、盈利、现金流、指数纳入这些老问题不会消失，只是被巨大的需求暂时盖住。上市首日如果波动很大，它会影响后面几单mega IPO的折价和配售策略。

对亚洲也有读法。SpaceX自身不给亚洲市场一个简单买点，但它会让投资者去寻找卫星通信、PCB、材料、通信设备和电力链条里的proxy plays。今天早上如果A股、港股或台湾供应链出现跟风，不能只看涨幅，要问两件事：这些公司到底是不是SpaceX供应链，订单是否会因为上市融资而增加。没有订单验证的proxy行情，容易变成短线情绪交易。

这里还要加一层供应链约束。Reuters引用美中贸易全国委员会调查说，受影响企业里有76%正在转移或寻找中国以外的关键矿产供应。这个信息和SpaceX、国防、EV、半导体、数据中心都连得上。资本市场可以给空间AI和卫星网络很高估值，但硬件最终要落到矿产、加工、认证和交付周期。亚洲的相关股票如果只跟着IPO情绪跳，风险在于估值先动，订单和材料保障还没跟上。

再把中国和G7这条线补上。中国国家统计局公布的5月PPI同比升3.9%，环比升0.5%，工业生产者购进价格同比升5.8%；CPI口径只有1.2%左右。这个组合说明上游能源和原材料已经抬价，但消费端还没有强到能顺畅接力。制造业夹在中间，外需强时可以靠出口消化一部分成本，贸易摩擦升温时，利润率就会被挤得很快。

Reuters同日还写到，中国副总理张国清参加了马克龙主持的全球经济失衡视频会议，时间就在法国G7峰会前。欧洲现在担心的是第二次China shock，尤其是EV、电池、光伏、机械和电子制造这些行业的低价出口。中国则强调本国工业政策不能被直接等同于不公平补贴。对港股和A股，这不是一句外资回流就能解释的事。今天要看哪些公司能把上游成本转出去，哪些公司会被欧洲反补贴、关税、本地化生产要求重新定价。对IB来说，这会影响跨境并购、供应链融资和香港上市故事的风险折扣。

第四件事是盘后软件。Adobe做创意软件、PDF、营销云和企业体验管理，是AI冲击软件商业模式时最容易被拿来观察的公司之一。它公布Q2后，数据显示每股收益5.96美元，收入66.2亿美元，高于市场预期；同时上调全年展望。但股价盘后仍跌，大约5%到6%，原因包括CFO离职、CEO接班不确定，以及投资者对AI会不会削弱传统创意软件定价权仍然不放心。

这和昨天Oracle的盘后反应放在一起看，信息更清楚。Oracle证明AI infrastructure需求强，但资本开支让人紧张；Adobe证明AI功能可以带动收入，但软件护城河和管理层稳定性还要被折价。市场仍然愿意买AI，只是区分得更细：谁能把AI变成现金流，谁只是在把AI变成更高投入和更复杂的治理故事。对hedge fund来说，这里可能继续产生long-short机会：硬件、云、软件、数据中心和应用层，不再是同一个beta。

我会把今天的行动点收得窄一点。第一，看油价能不能守住昨晚回落，如果Brent很快回到93美元以上，昨晚的风险修复就会被打折。第二，看10年美债在4.45%附近有没有真实买盘，还是只跟随能源 headline。第三，看SpaceX首日交易的开盘溢价、换手和零售成交，这会影响后续AI和航天相关IPO的定价。第四，看Adobe盘后跌幅能不能被消化，因为它会告诉我们，软件股在AI时代到底要用成长股倍数，还是先按治理和现金流折价。

今天很难归成一个简单的risk-on日。它更像是市场拿到了一次喘息，但马上要把喘息分配给不同资产：油价和利率给指数托底，PPI和ECB限制估值，SpaceX测试ECM窗口，Adobe提醒软件股还没有完全摆脱AI压力。早盘做判断时，先别急着追所有反弹。先问这笔钱是在买和平溢价，买降息想象，还是买少数稀缺资产的上市故事。答案不同，持仓时间也不同。
